## 🚀 Projeto em destaque

### 🤖 SCRIPTO — Organização Inteligente de Conteúdo Técnico

O SCRIPTO é uma solução criada para organizar, classificar e recomendar conteúdos técnicos utilizando técnicas de Ciência de Dados e Inteligência Artificial.

A aplicação recebe documentos técnicos, identifica informações relevantes e disponibiliza os resultados em formato JSON para integração com outras aplicações.

### Minha contribuição

- Análise Exploratória de Dados;
- limpeza e transformação do dataset;
- tratamento de registros duplicados;
- criação de categorias para documentos;
- geração de embeddings com Sentence Transformers;
- busca por similaridade semântica;
- recomendação de documentos relacionados;
- classificação automática;
- cálculo de confiança da classificação;
- definição do contrato de saída em JSON;
- documentação de métricas e limitações;
- planejamento de integração com Java/JVM.
